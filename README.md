DESENVOLVIMENTO DE APLICAÇÕES ORIENTADA A COMPONENTES

Desenvolvimento de uma Aplicação de Teste com React


Utilizando React, sua missão é desenvolver uma aplicação de teste. Pode ser algo simples, como um "Hello, World", por exemplo. Fique à vontade para usar sua criatividade e explorar outras ideias semelhantes.


Para ajudá-lo(a) no processo, há vários tutoriais disponíveis na internet que explicam como preparar o ambiente para rodar o React e fornecem exemplos de aplicações simples, como o "Hello World".

Após desenvolver a aplicação de Teste, você deve postar um print de tela do resultado obtido e, em seguida, compartilhar a pasta com os arquivos da aplicação através do Google Drive (para o email caroline.rosa@ulbra.br) ou compartilhar o link do GitHub com os arquivos da aplicação. O repositório no GitHub precisa ser público para que eu possa visualizar os arquivos.


Em caso de dúvidas durante o processo, não hesite em me acionar nos fóruns do ambiente Aula, por e-mail ou mensagem direta no ambiente Aula.

Estou aqui para ajudar e orientar você ao longo do desenvolvimento da atividade.
